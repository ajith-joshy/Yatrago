# YatraGo

YatraGo is an online bus ticket booking system developed using ASP.NET Core MVC and SQL Server.

# Features

- User registration and login
- Quick login functionality
- Search buses by source and destination
- Seat selection
- Passenger details entry
- Ticket booking and cancellation
- Ticket generation
- User profile management
- Admin dashboard
- Manage buses, routes, users, and bookings
- Reports and revenue analysis
- Live buses section
- Reset Password(only for users)

# Technologies Used

- ASP.NET Core MVC
- C#
- SQL Server
- ADO.NET
- HTML
- CSS
- JavaScript
- jQuery

# Database

The project uses SQL Server with stored procedures and normalized tables.
